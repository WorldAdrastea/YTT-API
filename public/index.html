Youtaite API

Welcome to the YTT API, this page will show the routes and queries for the CRUD API and give directions on how to use the API.
Endpoint:
Request Method:
What does it do?
/auth/login
POST
Allows a user to log in with email and password
/auth/register
POST
Allows for registration with the account requiring a username, email and password
/users
GET
Returns all existing users
/users/:id
DELETE
Allows a user to delete their account, only if the users' id matches the account their own account (to prevent other people deleting other people's accounts)
/users/:id
PATCH
Allows a user to update their information such as username, email or password
/uploads
GET
Returns all uploads
/uploads
POST
Allows a user to post an upload which requires a title, who the cover is covered by and original artists of the song
/uploads/:id
DELETE
Allows a user to delete their own upload
/uploads/:id
PATCH
Allows a user to update their own cover information
